Page 1: Escape Characters

Javascript Strings can use either single quotes 'string' or double quotes "string" to enclose the text, as long as they match.

If you want to include a quotation mark in the string, though, you must use a backslash escape character, \. You enter the \ before any character that has a special function in JavaScript to make the interpreter ignore the function of the character.

The syntax to display the string 'Don't run away' in Javascript is:
'Don\'t run away'.
