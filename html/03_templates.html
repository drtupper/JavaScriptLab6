Page 3: Templates

Template Literals are useful for interpolating variables and expressions into strings. Template Literals use backticks ` instead of quote marks. The syntax for this is:

let firstName = "Jane"

console.log(`Hello, ${firstName}!`);

Will display Hello, Jane! in the console. Note how the value saved in the variable firstName is inserted into the string.
